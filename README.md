## Running locally

Follow the steps there to setup a supabase project:
https://docs.embedbase.xyz/self-hosting/deployment


```yaml
# config.yaml
openai_api_key: ...
openai_organization: ...
log_level: DEBUG
environment: development
vector_database: supabase
supabase_url: ...
supabase_key: ...
```

```bash
make install
make run
```
